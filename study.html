今回の学び　ダイジェストVer

jQuery

ジョン・レシグが開発し、2006年1月にリリースした「JavaScriptのライブラリ」
基本構文
$(“セレクタ”).メソッド(“パラメータ[引数]”);

Pointは、セレクタに操作対象とするHTMLの要素、メソッドには処理内容を記載

＜script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/3.0.0/jquery.min.js"> //jQueryファイル

PointはHTMLのjQueryファイル引き当て、jsファイルの「.vegas」と画像フォルダの列挙



固定メニューバー

表示設定　例
HTML
＜div id="header_menu">
＜a href="URL">ホーム
＜a href="URL">サービス案内
＜a href="URL">施設のご紹介
＜a href="URL">ブログ
＜a href="URL">採用情報
＜/div>


CSS
#header_menu{
display: block;
position: fixed; /*ポジションを固定する*/
top: 0px;　/*一番上に設置*/
left: 0px;
z-index: 9999; /*どの要素よりも上にくるように*/
background-color: rgba(110,169,36,0.78);　　/*メニューバー背景色*/
width: 100%; /*横幅を100%に*/
padding: 0 0;
}


PointはHTMLのメニューバー宣言と、CSSファイルの「position: fixed」
